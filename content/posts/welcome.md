很久以来一直想要一个属于自己的写作空间。社交媒体太碎，朋友圈太私，微信公众号又有太多不必要的仪式感。所以最终选择了最朴素的方式 —— 一个静态博客。

## 为什么写

写作首先是为了思考。把一个想法从脑子里拽出来、用文字钉在纸上，往往才会发现自己其实没真的想清楚。Paul Graham 有句话我很喜欢：

> 如果你不写下来，你就不知道自己在想什么。

其次才是分享。如果其中有一两篇恰好对某个陌生人有用，那就是意外之喜。

## 会写什么

- **技术笔记** —— 工作中遇到的问题、踩过的坑
- **读书思考** —— 不写书评，只写让我有所触动的片段
- **生活随笔** —— 旅行、做饭、看电影、闲逛

## 关于这个站点

这个站点用 [Hugo](https://gohugo.io/) 搭建，部署在 GitHub Pages 上。设计风格是我自己写的 —— 深色 + 暖橘点缀，希望读起来不刺眼。

想要写新文章的时候，只需要在 `content/posts/` 目录下新建一个 `.md` 文件，git push 上去，几秒钟之后就会自动部署上线。

```bash
hugo new posts/my-new-post.md
git add .
git commit -m "new post"
git push
```

就这么简单。
